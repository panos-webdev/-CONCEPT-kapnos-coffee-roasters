KÁPNOS — Coffee RoastersA premium, conversion-focused brand website for a specialty coffee roaster. Designed and hand-coded from scratch with a sales-driven approach: every section is built to guide visitors toward an action — order beans, get directions, or visit the roastery.🔗 Live Demo
kapnos-roasters.netlify.app✨ Features

Interactive roast selector — visitors choose Light, Medium, or Dark and the content updates live
Auto-highlighting opening hours — the current day is highlighted automatically
Conversion-focused layout — clear calls-to-action and pricing throughout
Fully responsive — mobile-first design that works across all devices
Smooth scroll & subtle animations for a polished feel
Accessible — keyboard navigation and reduced-motion support
SEO-friendly — semantic HTML and Open Graph meta tags for clean link previews
